1. Iniciar servidor dos exercícios: npm run exercicios.
2. Iniciar servidor qrcode: npm run qrcode.
	Documentação qrcode: https://github.com/neocotic/qrious
3. Iniciar server sales: npm run sales


"scripts": {
		"exercicios": "http-server ./exercicios -c-1",
		"qrcode": "http-server ./qrcode -c-1",
		"sales": "http-server ./sales -c-1"
	},

4. Para o projeto calculadora, instalar:
	a) npm i -g @vue/cli

	b) Instalar plugin: Vetur

	c) local de extensões do vsc: C:\Users\Alex\.vscode\extensions

5. Iniciar projeto calculadora:
	npm run serve
