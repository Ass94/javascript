1. Download mongodb: 
    a) google: mongodb download
    b) aba: server
    c) clicar em download.
2. Instalar mongodb:
    a) selecionar completo.
    b) next, next, install.
3. Criar diretório no disco c.
    a) nome da pasta: data
    b) dentro da pasta data, criar pasta db (Local onde o mongo irá carregar os arquivos)
4. Em:
c: > Arquivos de Programas > MongoDB > Server > 4.2 > bin > mongod.exe (clicar duas vezes)
Após clicar duas vezes, verificar em qual porta o mongo estará executando: Port: 27017

5. Configurar variável de ambiente MONGO_HOME: C:\Program Files\MongoDB\Server\4.2\bin
6. Utilizar mongo no cmd: Abrir CMD e digitar mongo.
