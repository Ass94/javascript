1. criar projeto VueJS, nome: frontend
2. Inserir dependências no package.json
    C:\Softwares\Java Script\Projetos\exercicios-web\Projeto-Final\versao-inicial\frontend> npm install
3. executar projeto frontend: npm run serve
4. Fonte Lato referenciada dentro de public >> index.js: <link href="https://fonts.googleapis.com/css?family=Lato" rel="stylesheet">
5. dentro de src criar a pasta config.
6. dentro da pasta config criar arquivo store.js (área de armazenamento para compartilhar entre os componentes).
7. vuex: compartilhar dados dentro de uma aplicação.
8. dentro da pasta config criar arquivo bootstrap.js 
9. z-index: 1 (ter preferência. Aparecer antes / em cima dos demais componentes)
10. dentro da pasta components criar pasta home.
11. dentro da pasta home, criar Home.vue
12. na pasta src criar arquivo global.js
13. na pasta home criar arquivo Stat.js (Estatística.)
